🚀 Quick Start Guide

📋 Prerequisites

  • Node.js: 18.0+ (LTS recommended)
  • PostgreSQL: 14+ with extensions
  • Redis: 6+ for caching and sessions
  • Docker: 20+ with Docker Compose
  • Kubernetes: 1.25+ for production deployment

⚡ Development Setup

# 1. Clone the repository
git clone https://github.com/your-org/universal-ai-customer-service.git
cd universal-ai-customer-service

# 2. Install all dependencies
npm run install:all

# 3. Setup environment configuration
cp .env.example .env
# Edit .env with your specific configuration

# 4. Start infrastructure services
docker-compose up -d postgres redis rabbitmq

# 5. Run database migrations
npm run db:migrate

# 6. Seed initial data
npm run db:seed

# 7. Start all services in development mode
npm run dev

🐳 Docker Development

# Build and start all services
npm run docker:build
npm run docker:up

# View logs
npm run docker:logs

# Stop services
npm run docker:down

☁️ Production Deployment

# Deploy to Kubernetes
kubectl apply -f infrastructure/kubernetes/

# Or use Helm charts
helm install universal-ai-cs ./infrastructure/helm/

# Monitor deployment
kubectl get pods -n universal-ai-cs
