Easy Implementation

Simply include the universal components script and the navigation and footer will be automatically generated

Implementation Steps:

  1. Include Font Awesome and Google Fonts in your HTML head
  2. Add the universal-components.js script before closing body tag
  3. Remove existing navigation and footer HTML from your pages
  4. The components will automatically inject responsive navigation and footer
  5. Enjoy consistent, maintainable navigation across your entire site!
